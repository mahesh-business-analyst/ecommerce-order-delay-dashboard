# E-commerce Order Delay Dashboard

## Overview

Business Analyst project built in Power BI to analyze order delivery delays, identify bottlenecks, and recommend actions.

## Tools Used

* Power BI
* DAX
* Excel

## Key Features

* KPI Cards
* Delay Trend Analysis
* City & Courier Performance
* Delay Reasons
* Recommendations Section
